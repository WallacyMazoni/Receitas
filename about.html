Trabalho da Faculdade

Este é meu primeiro trabalho da faculdade, estou cursando Engenharia de Software e Desenvolvimento Web Full-Stack na Estácio.

Iniciei os estudos a pouco tempo, e já gosto muito da área! Este trabalho é o meu primeiro site.
Sei que não ficou muito profissional, mas ainda tenho muita coisa para aprender.


Se trata de um site que utiliza apenas HTML, CSS e um pouco de Bootstrap (é o conteúdo do trabalho).

Utilizei como material de estudo: os conteúdos disponíveis no SAVA, Vídeos do canal CFB Cursos, conteúdos do canal Rafaella Ballerini,
além de conteúdos estudados na plataforma Danki Code.



Recomendo todas estas fontes de pesquisa para quem interessar. Para fins de comparação, eu estou estudando Desenvolvimento Web há apenas 3 semanas.
Eu já programo em Pascal, Python, C++ e C#, mas este é meu primeiro contato com o desenvolvimento web e estou contente com o primeiro resultado.

Desenvolvi projetos pessoais, como um protótipo de um sistema ERP com funcionalidades BEM primitivas e realizei pequenos testes com programas funcionais utilizando aquela famosa cara de CMD.
Realizei projeto em grupo pela primeira vez utilizando o GIT e o GITHUB para auxiliar no trabalho colaborativo e no versionamento de código. Com o desenvolvimento de uma aplicação com o intuito de desenvolver um método de criptografia baseado em conceitos antigos porém utilizando características que adiciona um certo grau de funcionalidade. Projeto ainda em desenvolvimento em breve mais informações no meu portfólio que esta em desenvolvimento.
Fiz projetos pessoais na Engine Unity. Testei o uso do Windows Forms para a criação de aplicações Desktop e utilizei o VS Comunity com o mesmo propóstio. Atualmente nesta linguagem estou estudando o desenvolvimento de API´s para a utilização em projetos futuros.
